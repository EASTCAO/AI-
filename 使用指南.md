# AI产品拍摄智能评分系统（通义千问版）

## 快速开始

### 方法1：直接打开（推荐）
直接双击 `index_standalone.html` 文件，在浏览器中打开即可使用

### 方法2：使用启动脚本
双击 `启动评分系统.bat` 文件

## 首次使用设置

1. **获取免费API Key**
   - 访问：https://bailian.console.aliyun.com/#/api-key
   - 注册阿里云账号（新用户免费）
   - 获取API Key（格式：sk-xxxxxx...）
   - 免费额度：100万tokens（约5000次调用）

2. **配置API Key**
   - 打开网页后，在顶部看到"设置通义千问 API Key"
   - 粘贴您的API Key
   - 点击"验证并保存"
   - 看到"API Key验证成功"即可使用

## 使用步骤

1. **上传图片**
   - 点击或拖拽图片到上传区域
   - 支持一次上传多张图片（最多20张获得最佳效果）

2. **填写基本信息**
   - 摄影师姓名
   - 任务名称
   - 变体数量和类型

3. **AI自动评分**
   - 点击"开始AI自动评分"按钮
   - 等待5-15秒，AI会自动分析
   - 查看评分结果和详细理由

4. **保存记录**
   - 确认评分后点击"保存结果"
   - 可以导出今日所有评分记录为CSV文件

## 评分标准

系统基于以下维度自动评分：
- **产品类型**：常规/大件/组合礼包/发光类/DIY手工
- **拍摄难度**：简单/正常/较难/高难度
- **图片数量**：总图片数
- **场景图**：场景图数量及质量
- **图片质量**：产品质感、背景处理、画面构图
- **特殊处理**：造型、拼装、步骤图、多角度合成等

积分范围：0.25分 - 5分（以0.25为单位）

## 常见问题

### API Key验证失败
- 检查API Key是否正确复制（格式：sk-开头）
- 确认账号有剩余额度
- 检查网络连接是否正常

### 图片上传失败
- 单张图片不要超过10MB
- 支持格式：jpg, png, gif, webp, bmp
- 系统会自动压缩超大图片

### AI分析失败
- 检查是否已设置有效的API Key
- 确认账号额度未用完
- 如提示频率限制，请等待1分钟后重试

所有数据保存在浏览器本地存储（localStorage）：
- API Key：加密保存
- 评分记录：JSON格式保存
- 清除浏览器数据会删除所有记录（导出CSV备份）

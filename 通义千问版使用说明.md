# 通义千问版快速入门指南

## 🎯 版本说明

本版本已切换到 **阿里云通义千问 (Qwen-VL)** 多模态大模型，特点：

✅ **国内访问快** - 无需科学上网，阿里云节点
✅ **免费额度大** - 新用户赠送 100万tokens（约5000次调用）
✅ **图片支持好** - 支持最多20张图片同时分析
✅ **稳定性高** - 阿里云官方API，企业级服务质量

## 📝 使用步骤

### 1️⃣ 获取 API Key

1. 访问：https://bailian.console.aliyun.com/#/api-key
2. 登录/注册阿里云账号（支持支付宝快速登录）
3. 点击"创建 API Key"
4. 复制生成的 API Key（格式：sk-xxxxxx...）

**注意**：新用户会自动获得100万tokens免费额度！

### 2️⃣ 打开系统

双击 `打开通义千问版.bat` 或直接打开 `index_standalone.html`

### 3️⃣ 配置 API Key

1. 在页面顶部找到"设置通义千问 API Key"
2. 粘贴您的 API Key
3. 点击"验证并保存"
4. 看到"✅ 已就绪"表示配置成功

### 4️⃣ 上传图片并评分

1. **上传图片**：拖拽或点击选择一套产品的所有照片（最多20张）
2. **填写信息**：输入摄影师、任务名称、变体数量
3. **开始评分**：点击"🔍 开始AI自动评分"
4. **等待结果**：AI分析通常需要5-15秒
5. **保存记录**：查看评分后点击"保存结果"

## 🔧 常见问题

### Q1: 提示"API Key无效"？
- 检查是否从正确的页面复制（百炼平台）
- API Key格式应为 `sk-` 开头
- 确认账户状态正常（未欠费）

### Q2: 图片分析失败？
- 单张图片不要超过10MB（系统会自动压缩）
- 建议每次不超过20张图片
- 检查网络连接是否正常

### Q3: 请求过于频繁？
- 免费版有速率限制
- 建议每次请求间隔10秒以上
- 如需大量使用，可考虑付费版

### Q4: 如何查看剩余额度？
访问：https://bailian.console.aliyun.com/ → 费用中心 → 用量明细

## 💡 技巧建议

1. **批量处理**：每套产品单独评分，不要混合上传
2. **图片质量**：建议图片分辨率1080p以上，AI分析更准确
3. **变体设置**：正确选择变体类型（普通/颜色），系统会自动折算
4. **定期导出**：每日结束后导出CSV记录，避免数据丢失

# 版本更新日志

## V1.0 (2026-01-08) - 初始版本

### ✨ 新增功能
- 基于阿里云通义千问Qwen-VL的智能评分系统
- 支持单张图片和批量文件夹评分
- 二维评分体系：图片数量 + 产品难度
- 自动解析文件夹名称提取摄影师和任务信息
- 图片懒加载优化，支持大量图片高效预览
- 本地数据存储（API Key + 评分记录）
- CSV格式数据导出，支持Excel打开
- 沙漏加载动画
- 版本号标识系统

### 📋 评分标准
- **基准分**：0.85分 = 标准一套图（15-20张有效图，正常难度）
- **维度1**：图片数量调整（公式：(有效图-17.5)/30）
- **维度2**：产品难度调整（-0.2到+0.25分）
- 严格的图片折算规则（同角度细节、重复拍摄等）

### 🎨 UI/UX优化
- 响应式设计，支持大屏显示
- 现代化渐变色主题（紫色系）
- 流畅的过渡动画
- 友好的错误提示和加载状态
- 可折叠的参考速查表

### 📝 文档
- `README.md` - 项目说明文档
- `使用指南.md` - 详细使用手册
- `通义千问版使用说明.md` - 快速入门指南

### 🔧 技术实现
- 纯HTML/CSS/JavaScript（无依赖）
- 单文件应用，零部署
- LocalStorage数据持久化
- Intersection Observer API实现图片懒加载

## 维护指南

### 如何更新版本号

更新版本时，需要修改以下3个位置：

1. **页面标题** (index_standalone.html 第6行)
   ```html
   <title>AI产品拍摄智能评分系统 VX.X</title>
   ```

2. **Header徽章** (index_standalone.html 约第1467行)
   ```html
   <span class="version-badge">VX.X</span>
   ```

3. **页面底部** (index_standalone.html 约第1760-1762行)
   ```html
   <span class="version-number">VX.X</span>
   <span>更新日期：YYYY-MM-DD</span>
   ```

### 版本号规则

采用语义化版本号：`V主版本.次版本.修订版本`

- **主版本号**：重大架构变更、不兼容更新
- **次版本号**：新功能、功能优化
- **修订版本号**：Bug修复、小优化

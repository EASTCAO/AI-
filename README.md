# AI产品拍摄智能评分系统 V1.0

基于阿里云通义千问多模态大模型的产品摄影自动评分系统。

## 项目简介

本系统使用Qwen-VL视觉大模型，自动分析产品照片并给出客观评分，帮助摄影师和管理者快速评估拍摄工作量。

### 核心特性

- ✅ **零部署**：纯HTML单文件应用，双击即用
- ✅ **智能评分**：基于图片数量和产品难度的二维评分体系
- ✅ **批量处理**：支持文件夹批量评分，自动解析摄影师信息
- ✅ **图片懒加载**：支持大量图片预览，性能优化
- ✅ **数据导出**：CSV格式导出，支持Excel打开
- ✅ **本地存储**：API密钥和评分记录安全保存在浏览器

## 评分标准

**评分范围：0.1 ~ 3.0 分**，核心逻辑是估算拍摄耗时，再映射到对应分值（1分 ≈ 1.3小时）。

| 耗时估算 | 对应分值 | 典型场景 |
|--------|--------|--------|
| ≤5 分钟 | 0.10 | 极简单，1-2张 |
| 10~20 分钟 | 0.25 | 简单，含1张场景图 |
| 40~60 分钟 | 0.75 | 偏简单，低于标准 |
| ~1 小时 | 1.00 | 标准：15~20张，1~2场景 |
| ~1.5 小时 | 1.25 | 大件/需组装/金属重 |
| ~2 小时 | 1.50 | 礼盒套装/多件组 |
| ≥3 小时 | 3.00 | 发光产品/户外拍摄 |

**图片折算规则**：同角度细节3-4张折算为1张；1张场景图 ≈ 3张白底图；文件名含"备选/backup"的图片完全排除。

**AI合成图**：文件名格式为 `编号-摄影A图-开发以确认`，每张加 +0.1 分。

**批量变体折扣**：颜色变体5-8折，系列产品3-8折，由用户确认折扣比例。

## 技术栈

- 纯 HTML/CSS/JavaScript（无框架）
- 阿里云通义千问 Qwen3.5-Plus 多模态模型
- Node.js + Express（后端代理模式）
- LocalStorage 数据持久化
- Canvas API 图片压缩
- Intersection Observer API 图片懒加载

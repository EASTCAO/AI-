# AI产品拍摄智能评分系统 V1.0

基于阿里云通义千问多模态大模型的产品摄影自动评分系统。

## 项目简介

本系统使用Qwen-VL视觉大模型，自动分析产品照片并给出客观评分，帮助摄影师和管理者快速评估拍摄工作量。

### 核心特性

- ✅ **零部署**：纯HTML单文件应用，双击即用
- ✅ **智能评分**：基于图片数量和产品难度的二维评分体系
- ✅ **批量处理**：支持文件夹批量评分，自动解析摄影师信息
- ✅ **图片懒加载**：支持大量图片预览，性能优化
- ✅ **数据导出**：CSV格式导出，支持Excel打开
- ✅ **本地存储**：API密钥和评分记录安全保存在浏览器

## 评分标准

### 基准分：0.85分
标准一套图（15-20张有效图，正常难度）= 0.85分

### 评分公式
```
最终评分 = 0.85分（基准）+ 数量调整 + 难度调整
```

### 维度1：图片数量
- 数量调整 = (有效图片数 - 17.5) / 30
- 严格折算重复细节图和同角度拍摄

### 维度2：产品难度
- 简单产品：-0.15到-0.2分
- 正常产品：0分
- 有难度产品：+0.1到+0.15分
- 复杂产品：+0.15到+0.25分
